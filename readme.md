# Task Management Dashboard

A responsive Task Management Dashboard built using HTML, CSS and JavaScript.

## Features

- Add Tasks
- Edit Tasks
- Delete Tasks
- Search Tasks
- Dark Mode
- Local Storage
- Responsive Design
- Dashboard Statistics

## Technologies Used

- HTML5
- CSS3
- JavaScript

## Project Preview

A modern dashboard interface for managing daily tasks with priority levels, due dates, and progress tracking.
